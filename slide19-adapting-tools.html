Adaptations for Cognitive Needs

One-Page Profile
Standard
Written bullet points about preferences
Adapted
  • Use photos instead of text
  • Simple yes/no choices
  • Visual symbols (happy/sad faces)
  • Family helps create it
Communication Chart
Standard
Grid with emotions and responses
Adapted
  • Large pictures with clear emotions
  • Familiar objects person can point to
  • Record non-verbal cues
  • Include routine preferences
Decision-Making Tool
Standard
Written options with pros/cons
Adapted
  • One choice at a time
  • Use real objects to show options
  • Allow time for processing
  • Repeat choices consistently
Relationship Circle
Standard
Names in circles showing closeness
Adapted
  • Photos of people instead of names
  • Use hearts for loved ones
  • Color coding for roles
  • Update with person present

Adaptations for Sensory Needs

One-Page Profile
Standard
Visual document with text
Adapted for Visual
  • Large print (18pt+) or Braille
  • High contrast colors
  • Audio version available
  • Tactile elements for key points
Communication Chart
Standard
Visual grid format
Adapted for Hearing
  • Visual cues emphasized
  • Written preferences clear
  • Sign language preferences noted
  • Vibration/light alerts included
Decision-Making Tool
Standard
Written/verbal discussion
Adapted for Dual Sensory
  • Tactile communication methods
  • Hands-on demonstration
  • Deafblind manual alphabet
  • Extended time for communication
Support Planning
Standard
Meeting with verbal discussion
Adapted
  • Interpreter present if needed
  • Loop system for hearing aids
  • Good lighting for lip reading
  • Written summary provided

Adaptations for Language & Cultural Needs

One-Page Profile
Standard
English language document
Adapted
  • Translated to preferred language
  • Cultural preferences included
  • Religious/dietary needs clear
  • Family naming conventions
Communication Chart
Standard
English phrases and concepts
Adapted
  • Key phrases in both languages
  • Cultural gestures noted
  • Gender preferences for care
  • Prayer/spiritual needs timing
Decision-Making Tool
Standard
Individual-focused choices
Adapted
  • Include family decision-makers
  • Professional interpreter used
  • Cultural context explained
  • Allow time for family discussion
Care Planning
Standard
Medical model focus
Adapted
  • Include traditional healing views
  • Respect cultural health beliefs
  • Gender-specific care options
  • Festival/religious observances

Adaptations for Digital Access Barriers

Digital Tools
Standard
App-based or online tools
Adapted
  • Paper versions always available
  • Phone call options
  • Face-to-face support offered
  • Library/community center help
Information Sharing
Standard
Email or patient portal
Adapted
  • Printed copies posted
  • Text message updates
  • Telephone reminders
  • Family member can access
Virtual Appointments
Standard
Video consultation expected
Adapted
  • Phone call alternative
  • In-person option maintained
  • Support to set up technology
  • Test call before appointment
Self-Management Tools
Standard
Smartphone apps for tracking
Adapted
  • Paper diaries provided
  • Simple charts and stickers
  • Family member can track
  • Regular check-in calls
Universal Principles for Adaptation
Ask don't assume
Co-create with person
Regular review & update
